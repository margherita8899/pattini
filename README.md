# pattini

Sito semplice dedicato al mondo del pattinaggio sul ghiaccio.

## Caratteristiche

- **Homepage** con immagine di copertina e titolo animato.
- **Storia e origini** del pattinaggio sul ghiaccio.
- **Tipi di specialità** con descrizioni di artistico, velocità, hockey e sincronizzato.
- **Galleria** con molte immagini di diverse discipline.
- **Animazioni**: un piccolo skater si muove attraverso lo schermo e un titolo macchina da scrivere.
- **Sezione suoni** con un lettore audio incorporato.
- **Consigli** per principianti con un elenco di buone pratiche.

## Uso

Apri il file `index.html` in un browser per visualizzare il sito. Tutte le risorse sono caricate da URL esterni, quindi è necessaria una connessione a Internet.
